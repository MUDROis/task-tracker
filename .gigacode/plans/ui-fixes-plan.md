### 3. Задачи с высоким приоритетом не попадают в раздел "Срочные"
**Файл**: `app.js`, функция `renderBoard`  
**Цель**: Проверить и исправить логику автоматического перемещения задач с high приоритетом в 'urgent'

**Текущая логика** (строка ~382):
```javascript
// Если задача имеет высокий приоритет, автоматически ставим её в 'urgent'
userTasks.forEach(function(t) {
    if (t.priority === 'high' && t.status !== 'urgent') {
        t.status = 'urgent';
    }
});
```

**Проверить**:
- Значение `t.priority` (должно быть 'high')
- Значение `t.status` (должно быть ≠ 'urgent')
- Сохраняется ли изменение в базу данных

**Возможное решение**:
```javascript
userTasks.forEach(function(t) {
    if (t.priority === 'high' && t.status !== 'urgent') {
        t.status = 'urgent';
        saveTask(t); // Сохраняем изменение в базу
    }
});
```

### 4. Всплывающее окно выбора даты не адаптируется
**Файл**: `style.css`  
**Цель**: Сделать модальное окно адаптивным для мобильных устройств

**Исправление**:
- Изменить `max-width: 500px` на `max-width: 90vw` в `.modal-content`
- Добавить адаптивность для мобильных экранов

```css
.modal-content {
    max-width: 90vw;
    width: 90vw;
}

/* Адаптивность для маленьких экранов */

    .modal-content {
        width: 95vw;
        max-width: 95vw;
    }
}
```
